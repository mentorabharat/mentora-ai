# Classroom Learning Blocks

Each classroom can contain the following blocks:

## Core Blocks (Always Available)
1. Text Block
   - Concept explanation
   - Definitions
   - Structured notes

2. Explanation Block
   - Step-by-step reasoning
   - Examples
   - Exam-oriented clarity

3. Practice Block
   - Questions
   - Exercises
   - Practical tasks (especially IT)

## Optional Blocks (Phase-Based)
4. Interactive Block
   - Click-based learning
   - Decision paths
   - Scenario understanding

5. Video Block (Optional)
   - Pre-recorded
   - Short duration
   - Concept support only

6. Live Block (Future Phase Only)
   - Mentor interaction
   - Doubt clearing
   - NEVER default

Rule:
Text + Explanation + Practice = Classroom Complete
