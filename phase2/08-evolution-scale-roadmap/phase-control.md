# Phase Control System

Mentora World AI follows strict phase discipline.

Rules:
- Each phase has a defined scope
- No phase overlaps
- No backward dependency
- Phase-1 = base
- Phase-2 = structure
- Phase-3+ = execution only

Phase activation:
- Manual approval required
- Founder authority mandatory
- AI cannot activate phases

Goal:
Prevent uncontrolled growth and vision drift.
