# IT Learner Journey

Entry Intent:
"I want to build / understand / practice"

Journey Flow:
1. Skill intent selection
2. Basic theory grounding
3. Practical exposure
4. Code interaction
5. Error understanding
6. Iteration or exit

System Principles:
- Theory is minimal but mandatory
- Practice is central
- No copy-paste learning
- AI assists, does not code blindly

AI Roles:
- Practice Agent
- Explanation Agent
- Analytics Agent

Growth Signal:
- Time spent in practice
- Re-attempts
- Project completion

Not Allowed:
- Certificate selling
- Fake job guarantees
