# Student Journey (General Education)

Entry Points:
- Search engine
- Recommendation
- Shared link
- Language-specific access

Journey Flow:
1. Discovery
2. Topic selection
3. Concept learning
4. Example understanding
5. Practice exposure
6. Exit OR continuation

Key System Rules:
- No forced signup
- No false promises
- Depth increases gradually
- AI explains, does not decide future

AI Roles Involved:
- Content AI
- Explanation AI
- Language AI

Human Control:
- Content approval
- Policy boundaries

Success Signals:
- Time spent
- Return visits
- Topic progression

Failure Signals:
- Immediate exit
- Repeated confusion
